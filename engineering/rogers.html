At Rogers, I was a co-op student in the R&D department working on manufacturing and testing novel ferrite materials. I manufactured novel ferrite materials for radio frequency applications using ceramic processing techniques. I also tested various series using VNA, analyzed resulting data through Python scripts and Uncountable application.

Additionally, I updated GUI of existing Python scripts with additional features to streamline use. I also assisted in general lab upkeep including rewriting documentation to reflect current procedures and using SolidWorks to model 3D printed parts to facilitate various tasks.
